# Citing Neural Representation Explorer

If this repo was useful for your research, teaching, blog post, or any other work, please cite it. A GitHub star is also nice.

## Short form

> Graham, P. (2026). *Neural Representation Explorer: recovering behavioral state structure from neural population activity with PCA, UMAP, and K-Means*. GitHub. https://github.com/peterajhgraham/neural-representation-explorer
